# Smart Hospital Appointment & Diagnostic System

A production-ready backend application built using Java 17, Spring Boot 3, Spring Security, JWT, MySQL, Apache Kafka, Docker, and JUnit.

## Tech Stack

- Java 17
- Spring Boot
- Spring Security
- JWT
- MySQL
- Apache Kafka
- Docker
- Swagger
- JUnit
- Mockito
